# Lantern Lighter

**Lantern Lighter** is a prototype survival game built around a unique mechanic:  
as time passes, the player’s vision gradually fades, making navigation and survival increasingly difficult.

---

## Tools:
I used the following tools to make this prototype:
- Godot/Gdscript
- Adobe Illustrator

---

## How to Run

**WINDOWS:**
1. Download **LanternLighter.exe** in WINDOWSEXE folder
2. Launch the executable to start playing

**MAC:**
1. Download and extract the zip file in MAC folder
2. Launch the executable to start playing

**LINUX:**
1. Download and launch files in LINUX folder
2. Run files to start playing
---

## Controls

- **W / A / S / D** — Move
- **Left Click** — Shoot arrows

---

Survive as long as you can while the darkness closes in.
